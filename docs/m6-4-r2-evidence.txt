titles). The DIGITS expansion is back to 10 cells (0..9 at 36° each); the

Manual touch protocol (verified by user in sim):
  1. Tap DIGITS wedge → expansion now shows ONLY 0..9 (no ", ', - cells).
  2. Type שבק → results include שב"ק because Search._normalize strips
     the gershayim during matching. The user never types ".
  3. Type שיר השירים (space, not hyphen) → results include שיר-השירים-המלא
     because Search._normalize treats the hyphen as a space during matching.
  4. Article-open flow + long-press flow unchanged.
